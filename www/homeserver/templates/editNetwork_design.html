Netzwerkeinstellungen
Diese Einstellungen werden verwendet, um z.b. für das Onlineupdate eine Verbindung zum Server aufbauen zu können.
Ist kein Proxy-Server vorhanden, sollte das Feld leer bleiben.
Ausserdem ist es möglich die Hausautomatisierung in einem separaten Netzwerksegment laufen zu lassen.
Dazu einfach die IP des Segments und die dazugehörige Netzmaske eintragen.
Der Port des Hausbus-Netzwerks sollte erst verändert werden, wenn alle Controller im Netzwerk bereits auf diesen Port umgestellt sind.
Controller->Ethernet->setConfiguration( IP, Port )
Andernfalls werden diese Controller anschließend nicht mehr erreichbar sein.
